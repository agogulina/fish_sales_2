# Данные 

| Файл | Что это |
|------|---------|
| `sales.csv` | все продажи помесячно: `Головное СКЮ Артикул`, `Головное СКЮ Наименование`, `Категория`, `Группа new`, `УПП__Группа клиентов`, `Дата`, `Кол_шт` |
| `promo_sales1.csv` | промо-факт продажи (та же структура) |
| `promo_final.csv` | недельные скидки: `Головное СКЮ Артикул`, `УПП__Группа клиентов`, `_промо план шт`, `_промо %скидки`, `Год`, `Неделя`, `Дата начала недели` |
| `lenta_prices.xlsx` | прайс розничной сети (для Слоя 3): наименование + цена входа/полки за штуку. Индексы столбцов настраиваются в `src/pricing.py:load_lenta_prices`. |

Продажи помесячные; в коде агрегируются по месяцам, объём логарифмируется (`log1p`).
